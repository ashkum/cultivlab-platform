# Your CultivLab Site 🌱

Welcome! This folder has the starting code for **your own website**.

Your website is live on the internet at the URL we gave you (something like
`https://l07.cultivlab.com`).

---

## What's in this folder?

- **`index.html`** — Your home page. This is the file you'll edit most.
- **`README.md`** — This file. You're reading it now.

That's it for now! As you learn more, you'll add more files like:

- `about.html` — a page about you
- `style.css` — a file just for colors and styles
- `projects.html` — a page showing what you've built

---

## How to edit your site

1. **Open `index.html`** in your editor.
2. **Look for the arrows that say `← CHANGE THIS`** — those are spots that are easy and fun to
   change.
3. **Save the file** (Ctrl-S or Cmd-S).
4. **Refresh your website** in your browser — your changes will be live!

---

## Easy things to try first

### 1. Change the title

Find this line in `index.html`:

```html
<h1>Hi, I'm a CultivLab student!</h1>
```

Change `Hi, I'm a CultivLab student!` to your own greeting, like:

```html
<h1>Hi, I'm Alex and I love dinosaurs!</h1>
```

### 2. Change the colors

Find the part that says `CHANGE COLORS HERE` near the top of the file. Try changing the colors —
replace `#d97706` with `purple`, `pink`, `blue`, or any color name you know.

### 3. Change the button text

Find:

```html
<button onclick="sayHi()">Click me!</button>
```

Change `Click me!` to something else, like `Press for a surprise!`
